Algorithm Visualization

Backtracking

An algorithm traversing trees/graphs, prioritizing the exploration of as deep a path as possible before backtracking.

Depth-First Search

A problem solving technique that explores different possible solutions and "backtracks" to previous options if they are not viable.

Breadth-First Search

An algorithm for traversing trees/graphs, exploring all neighbour nodes at the present depth before moving on to a deeper level.

Binary Search

An algorithm for finding an element in a sorted list, repeatedly dividing the search interval in half until the target is located.
