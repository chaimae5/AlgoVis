# AlgoVis

AlgoVis is a web-based visualization tool for learning and exploring classic algorithms and data structures. It provides interactive visualizations for algorithms such as Backtracking, Binary Search, DFS, BFS, Bubble Sort, and Merge Sort.

## Features
- Visualize algorithm execution step-by-step
- Interactive tree and array visualizations
- Clean, responsive UI
- Easily switch between different algorithms

## Usage
1. **Open `AlgoVis.html` or any algorithm-specific HTML file in your browser.**
2. **Select or input your data as prompted.**
3. **Click 'Execute' to see the algorithm in action.**
4. **Explore the visualization and node values in the split view.**

## Project Structure
- `AlgoVis.html` — Main landing page
- `algorithms.html` — List of available algorithms
- `Backtracking.html`, `BinarySearch.html`, etc. — Algorithm-specific visualizations
- `script.js`, `scriptBacktracking.js` — JavaScript logic for visualizations
- `styles.css` — Main stylesheet
- `imgs/` — Algorithm illustration images

## Requirements
- Modern web browser (Chrome, Firefox, Edge, Safari)
- No installation required

## Customization
- Add new algorithms by creating a new HTML and JS file, then link it in `algorithms.html`.
- Update `styles.css` for custom themes or layouts.


**Enjoy learning algorithms visually with AlgoVis!**
